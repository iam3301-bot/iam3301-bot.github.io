1. **打开任意页面**

2. **找到主题按钮**

## 🐛 故障排除

### Q: 主题没有生效？
**A**: 检查以下几点
1. 确认浏览器支持（Chrome 88+）
2. 清除浏览器缓存
3. 检查控制台是否有错误
4. 尝试刷新页面

### Q: 切换主题有延迟？
**A**: 正常现象
- 首次切换加载字体（1-2秒）
- 后续切换即时生效（<100ms）

### Q: 某个页面主题不一致？
**A**: 不应该发生
- 所有页面自动同步
- 如果遇到请刷新页面
- 或重新选择主题

### Q: 移动端显示不正常？
**A**: 检查视口设置
- 所有主题已适配移动端
- 确保没有浏览器缩放
- 尝试横屏/竖屏切换

## 🚀 高级功能

### 开发者API

```javascript
// 获取主题管理器实例
const themeManager = window.globalThemeManager;

// 获取当前主题
const current = themeManager.getCurrentTheme();
console.log(current.name); // "黑神话：悟空"

// 切换主题
themeManager.applyTheme('witcher', true);

// 获取所有主题
const allThemes = themeManager.getAllThemes();

// 监听主题切换事件
window.addEventListener('globalthemechange', (e) => {
  console.log('主题已切换:', e.detail.theme.name);
});
```
