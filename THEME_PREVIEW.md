# GameBox 主题预览 🎨

## 如何使用主题切换功能

### 步骤1：找到主题切换器
在网页右下角，你会看到一个浮动的主题按钮：

```
┌─────────────────┐
│  🎨 主题        │  ← 点击这里
└─────────────────┘
```

### 步骤2：选择主题
点击按钮后，会弹出主题选择面板：

```
┌──────────────────────────────┐
│  🌃  赛博朋克                 │
│      未来科技风格，霓虹灯效果  │  ✓
├──────────────────────────────┤
│  🎮  游戏世界                 │
│      经典游戏风格，复古与现代  │
└──────────────────────────────┘
```

### 步骤3：享受新主题
选择后主题会立即应用，并自动保存你的偏好！
